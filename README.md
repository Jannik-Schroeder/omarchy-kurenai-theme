# Kurenai (紅) — Omarchy Theme

A deep crimson theme for [Omarchy](https://omarchy.com). I made this because there wasn't a good red theme — so here we are.

Dark backgrounds, warm reds, Japanese vibes. WCAG AA contrast ratios across the board.

## Install

```bash
omarchy-theme-install https://github.com/jannik-schroeder/omarchy-kurenai-theme.git
```

That's it. It clones the theme and applies it automatically.

## Switch to it later

```bash
omarchy-theme-set kurenai
```

## What's included

- `colors.toml` — core palette (drives terminal, hyprland, waybar, etc. via templates)
- `btop.theme` — system monitor
- `neovim.lua` — kanagawa-dragon with kurenai overrides
- `vscode.json` — VS Code theme
- `icons.theme` — Yaru-red icon set
- 5 backgrounds — crimson waves, red moon wave, fuji with autumn leaves, temple sunset, sakura temple
